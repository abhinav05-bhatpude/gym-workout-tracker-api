# Gym Workout Tracker API

RESTful backend API for tracking gym workouts built with Node.js, Express.js, MongoDB, and Mongoose.

## Features

- Create workout
- Get all workouts
- Get workout by ID
- Update workout
- Delete workout
- Filter workouts
- Search workouts
- Sort workouts

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- Postman

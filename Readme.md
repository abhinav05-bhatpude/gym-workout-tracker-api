# Gym Workout Tracker API

RESTful backend API for tracking gym workouts built with Node.js, Express.js, MongoDB, and Mongoose.

## Features

- Create workout
- Get all workouts
- Get workout by ID
- Update workout
- Delete workout
- Filter workouts
- Search workouts
- Sort workouts

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- Postman

## Project Status

## Live API

Base URL:

https://gym-workout-tracker-api-2.onrender.com

### Example Endpoints

- GET `/api/workouts`
- POST `/api/workouts`
- GET `/api/workouts/:id`
- PUT `/api/workouts/:id`
- DELETE `/api/workouts/:id`

> Note:
> The API is hosted on Render free tier, so the first request may take a few seconds.
